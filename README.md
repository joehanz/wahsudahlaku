# 📢 Iklan Baris Online

Platform sederhana untuk memasang dan mencari iklan baris secara gratis tanpa login.

---

## 🚀 Tentang Proyek

Website ini adalah sistem iklan baris berbasis GitHub Pages + Google Sheets, yang memungkinkan pengguna:

- Memasang iklan tanpa login  
- Menampilkan iklan secara otomatis  
- Menghubungkan penjual dan pembeli lewat WhatsApp  
- Berjalan tanpa backend server  

---

## ✨ Fitur Utama

- 📝 Pasang iklan melalui form sederhana  
- 🔎 Lihat daftar iklan secara real-time  
- 📱 Klik langsung chat WhatsApp  
- 📂 Kategori iklan (produk, jasa, dll)  
- ⚡ Gratis hosting di GitHub Pages  
- ☁ Data disimpan di Google Sheets  

---

## 🧩 Cara Kerja Sistem

User isi form iklan  
↓  
Google Apps Script  
↓  
Google Sheets (database)  
↓  
GitHub Pages mengambil data (JSON)  
↓  
Iklan tampil di website
